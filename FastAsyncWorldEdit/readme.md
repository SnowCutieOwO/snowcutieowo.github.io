## 关于插件

FastAsyncWorldEdit 是 WorldEdit 的分支，对其性能及内存占用进行了诸多改进，同时加入了许多新内容。如果你正在使用其他依赖 WorldEdit 的插件，安装 FAWE 会提升它们的性能。
