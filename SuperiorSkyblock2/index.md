SuperiorSkyBlock 2 是一款颇为流行的空岛插件。它内置多个模块（附属），允许玩家在体验原汁原味空岛的基础上通过任务和升级功能让自己的岛屿拥有更多获取资源的方式。除此之外，插件还开放了 API 供开发者接入。
